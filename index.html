Automação Financeira

Baixa Automática via CNAB + API

O Problema

Baixas de pagamento eram feitas manualmente, gerando erros, atrasos e retrabalho. O operador precisava conferir extrato e dar baixa um por um.

Minha Solução

  • Integração com arquivos CNAB (retorno bancário) para baixa automática em lote
  • Fallback via API de pagamento para confirmação em tempo real
  • Conciliação automática entre cobranças e pagamentos recebidos

Resultado

Processo que levava horas virou automático. Eliminou erros manuais e liberou o operador para tarefas estratégicas.

PHP/Yii2CNABAPI de PagamentoMySQL
Automação Fiscal

Emissão Automatizada de NF-e

Desenvolvido pela Larissa — contribuí com monitoramento e infraestrutura

O Problema

Emissão de notas fiscais era manual, propensa a erros de digitação e consumia tempo da equipe administrativa.

Solução (dev: Larissa) + Minha Contribuição

  • Integração com API de NF-e para emissão automatizada direto do sistema
  • Dados puxados automaticamente do contrato e do cadastro do cliente
  • Geração e armazenamento do XML e DANFE
  • Minha parte: monitoramento do sistema, monitoramento do banco de dados e configuração de crons

Resultado

Notas fiscais emitidas com um clique, sem retrabalho e sem erros de digitação.

PHP/Yii2NF-e APIXMLMySQLCronMonitoramento
Ferramenta

Gerador de Relatórios Personalizáveis

O Problema

Cada imobiliária precisava de relatórios diferentes — financeiros, de imóveis, de contratos — e ficavam dependentes do suporte para qualquer mudança.

Minha Solução

  • Engine de relatórios com filtros dinâmicos configuráveis pelo próprio usuário
  • Templates para diferentes tipos: financeiro, contratos, imóveis, repasses
  • Exportação em múltiplos formatos

Resultado

Usuários criam seus próprios relatórios sem depender do suporte. Reduziu chamados de relatórios drasticamente.

Projeto Independente

PDV Completo — Conveniência Rocá

Sistema de ponto de venda com gestão completa, integrações com iFood, Telegram e IA

PDV — Grid de produtos com carrinho
Mesas — Contas abertas e livres
Saúde do Negócio — Dashboard com IA
Estoque — Produtos com filtros e status
Vendas — Relatório com filtros por período
Clientes — Cadastro com saldo e histórico
Usuários — Admin e Operador com PIN
Categorias — Cervejas, Bebidas, Petiscos...

O Problema

Conveniência operava com controle manual: sem sistema de vendas, sem gestão de mesas, sem visibilidade financeira e sem integração com delivery.

Minha Solução

  • PDV completo com grid de produtos, carrinho, categorias por abas e cálculo automático
  • Gestão de mesas/contas com status em tempo real (abertas, livres, valor por mesa)
  • Controle de estoque com filtros, categorias, status ativo/inativo e alertas
  • Dashboard "Saúde do Negócio" com faturamento, ticket médio, margem e top produtos por lucro
  • Relatório de vendas com filtros por período (hoje, ontem, semana, mês)
  • Cadastro de clientes com histórico de compras e saldo
  • Gestão de usuários com níveis de acesso (Admin/Operador) e autenticação por PIN
  • Categorias personalizáveis (Cervejas, Bebidas, Destilados, Petiscos, Tabacaria, etc.)
  • Integração com iFood para recebimento automático de pedidos
  • Bot Telegram com IA para consultar saúde do negócio e emitir relatórios via chat

Resultado

Sistema completo em produção. Dono acompanha o negócio pelo celular via Telegram. Pedidos do iFood entram automaticamente. Decisões baseadas em dados reais, não em achismo.

Infraestrutura

Servidor de Produção do Zero

O Problema

Aplicações Next.js e Node.js precisavam rodar em produção com alta disponibilidade, SSL, múltiplos domínios e gerenciamento simplificado.

Minha Solução

  • Servidor Linux (Debian) configurado do zero
  • Nginx como proxy reverso para múltiplas aplicações
  • aaPanel para gerenciamento visual do servidor
  • Docker para isolamento de ambientes
  • SSL, backups e monitoramento

Resultado

Múltiplas aplicações rodando em um único servidor, cada uma com seu domínio, SSL e deploy independente.
